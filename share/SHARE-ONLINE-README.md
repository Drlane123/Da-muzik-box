# Da Music Box — Share Online

## Files

| File | Purpose |
|------|---------|
| `DA-MUSIC-BOX-FEATURE-GUIDE.html` | **Open in browser** — full feature page with images, copy buttons, routing diagram |
| `screenshots/` | Drop your PNG/JPG screenshots here (see README inside folder) |
| `../DA-MUSIC-BOX-PLAY-STORE-GUIDE.md` | Long-form markdown for Play Store / press |

## Quick start

1. **Double-click** `DA-MUSIC-BOX-FEATURE-GUIDE.html` to preview in Chrome/Edge/Firefox.
2. Take app screenshots and save them to `screenshots/` using the filenames in `screenshots/README-SCREENSHOTS.txt`.
3. Refresh the HTML page — images appear automatically.
4. Click **Copy for Reddit / Facebook** at the top for a ready-made social post (includes auto-tune + vocoder).

## Share online (pick one)

### Option A — Screenshot posts (Reddit / Facebook)
- Copy the social post text from the HTML page.
- Attach screenshots #03, #04, #05 (Vocal DSP, Auto-Tune, Vocoder) plus Beat Lab or Groove Lab.
- Post to r/WeAreTheMusicMakers, r/edmproduction, Facebook groups, etc.

### Option B — Host the HTML page (link post)
Upload the entire `share` folder to:
- **GitHub Pages** (free)
- **Netlify Drop** (drag folder → get URL)
- **itch.io** project page
- Your own website

Share the URL to `DA-MUSIC-BOX-FEATURE-GUIDE.html`. Facebook/Reddit will show a link preview if `screenshots/01-mastering-bay.png` exists.

### Option C — Google Drive / Dropbox
- Zip the `share` folder.
- Upload and share — recipients download and open the HTML locally.
- (Drive does not render HTML in-browser; they open the file after download.)
